# mt-website-server
素材网站服务器
基于nodejs + sequelize.js + mysql

#主要功能介绍
前端瀑布流图片展示，可预览下载图片，上传图片，收藏图片，管理员管理图片，图片分类等，动态记录用户的下载量和浏览量以及上传量
